This page renders a Heighway dragon curve, a type of fractal curve, in SVG graphics using recursive methods. A Heighway dragon curve can be generated by taking a line segment, "popping" it either in or out into two smaller segments joined by a 90 degree angle, then continually repeating this process on all subsequent segments, alternating the "popping" direction for each segment. A more mathematically rigorous explanation of this process can be found here. On this page, you can set the recursion depth, the number of times this process is repeated, as well as how the curve is styled.

Another way to generate the curve, commonly taught to young students, is by taking a long strip of paper, repeatedly folding it in half over itself, then unfolding the strip such that all corners are at 90 degree angles. In this case, the recursion depth is equivalent to the number of times the hypothetical paper strip has been folded.
